About this Project

This project prioritizes functionality over style and design, with the latter to be addressed in subsequent phases. The primary focus here is to develop a web application connected to a database, with the main purpose of presenting dynamic data retrieved from the database. The application is currently deployed and hosted on Render.com, while the database is hosted on the PlanetScale platform.

This project aligns perfectly with my current career goals, as I aim to transition into the IT field, specifically pursuing a career as a web/front-end developer. Working on this project has been both educational and enjoyable, as I've encountered numerous challenges and opportunities for learning. Even though the project is ongoing, it has already provided me with valuable insights and skills that I look forward to further refining."

What I Learned:

  • Flask: I gained a solid understanding of the fundamentals of Flask, a popular Python web framework.
  • Bootstrap: I became proficient in using Bootstrap for styling, including creating visually appealing navigation bars and footers.
  • Jinja: I learned how to leverage Jinja template tags to inject dynamic data into HTML templates, loop through dictionaries, and handle conditional logic effectively.
  • API: I successfully implemented an API route to provide JSON data.
  • Cloud: The project involved deploying my website to Render.com and setting up a cloud-based MySQL database on PlanetScale.com. I also learned how to manage sensitive connection credentials and SSL certificates to ensure secure database access.
  • Mailto: I integrated mailto: links for 'Apply' and 'Contact Us' buttons, constructing email templates and converting them into clickable URLs.
  • SQL: My experience with SQL included creating tables, inserting data, connecting my web app to the database using SQLAlchemy, and securely managing sensitive connection credentials.

Challenges Faced:

  • Database Management: Setting up and managing a cloud-based MySQL database, along with securely handling sensitive connection credentials, posed a significant challenge.
  • SSL Certificates: Managing SSL certificates to establish secure database connections added complexity to the project.
  • API Integration: Implementing an API route and understanding the intricacies of API integration required a solid grasp of API concepts.
  • Dynamic Data Handling: Effectively managing dynamic data, including iterating through dictionaries and handling conditional logic, presented its own set of challenges.
  • Deployment: Deploying the web app to Render.com and ensuring its functionality in a live environment demanded meticulous attention to detail.
  • Email Integration: Constructing mailto: links and email templates for 'Apply' and 'Contact Us' buttons involved a deeper understanding of email integration.

In conclusion, this project provided me with invaluable hands-on experience in web development using Python, Flask, and database-driven web applications. It also exposed me to a variety of tools and services crucial for modern web development.

Future Plans: In near future (from 25/09/2022) I plan to enhance this application by adding an admin section. In this admin section, administrators will have the capability to create, update, and delete job postings, as well as manage job applications by marking them as accepted or rejected. Furthermore, I intend to implement a feature that allows the uploading of files in formats such as xlsx and csv, facilitating the addition of multiple job ads to the database. Eventually, my goal is to transform this project into a JavaScript + ReactJS-based application.
